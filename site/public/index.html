🦜 squawk

Voices for your Claude Code agents on macOS. Talk to Claude hands-free — every agent in your cmux setup speaks with its own voice.

View on GitHub Quick start → 
git clone https://github.com/ashrocket/squawk && cd squawk
./setup.sh
./voice --user YourName   # speak when greeted

Fully local ears and voices. whisper.cpp (Metal) for speech-to-text, macOS neural voices or Kokoro for text-to-speech. Nothing leaves your Mac except what you already send to Claude.

No cloud STT, no cloud TTS, no extra API key. Runs entirely on your Mac.

Radio-operator rules for a crew of agents

Run voice in every cmux tab. The agents coordinate automatically — one channel, no chaos.

🎙️

Focus follows mic

Only the focused window's agent listens; the rest stand by quietly.

📻

Con-holding

An active exchange keeps the mic for 75 s past the last turn — nobody steals the channel mid-thought.

🗣️

One talker

A global speech lock. Agents queue; they never talk over each other.

🎭

Distinct voices

Every agent auto-gets its own voice — Kokoro neural, Apple Premium, or system default, best first.

📨

Relay requests

Background agents queue messages; the active conversation reads them aloud between turns.

🔤

Pronunciation learning

Say "pronounce cmux as sea mux" once — every agent and every voice learns it, in speech and hearing.

⚡

Barge-in

Start talking over the assistant and it stops to listen. Experimental, but it works.

🖥️

Squawk.app

Native macOS settings panel: one-click install checks, voice auditions, pool curation, lexicon editing.

🔇

Echo suppression

Talk while the agent is speaking and the speech lock prevents your voice from feeding back as input.

How it works

whisper.cpp listens, Claude thinks, your chosen voice speaks. One clean pipeline.

mic ──▶ voice detection ──▶ whisper.cpp (Metal) ──▶ claude -p --resume
                                                          │
speakers ◀── say / Kokoro ◀── lexicon ◀── speech lock ◀───┘
