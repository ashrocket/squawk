🦜 squawk

Voices for your Claude Code agents on macOS. Talk to Claude hands-free — and let every agent in your cmux setup speak to you, one at a time, each with its own voice.

View on GitHub Quick start 
git clone https://github.com/ashrocket/squawk && cd squawk
./setup.sh
./voice --user YourName   # speak when greeted

Fully local ears and voices. whisper.cpp (Metal) for speech-to-text, macOS neural voices or Kokoro for text-to-speech, the Claude Code CLI as the brain. No API key. Nothing leaves your Mac except what you already send to Claude.

Radio-operator rules for a crew of agents

🎙️ Focus follows mic

Run voice in every cmux tab. Only the focused window's agent listens; the rest stand by quietly.

📻 Conversations hold the con

An active exchange keeps the mic for 75s past the last turn — nobody steals the channel mid-thought.

🗣️ One talker

A global speech lock. Agents queue; they never talk over each other.

🎭 Distinct voices

Every agent auto-gets its own voice, best available first: system default, Kokoro neural, Apple Premium.

📨 Relay requests

Background agents queue messages the active conversation reads aloud between turns.

🔤 Agents learn pronunciation

Say "pronounce cmux as sea mux" once — every agent and every voice learns it, in speech and hearing.

⚡ Barge-in

Start talking over the assistant and it stops to listen (experimental).

🖥️ Squawk.app

A native settings panel: one-click install checks, voice auditions and pool curation, lexicon editing.

How it works

mic ──▶ voice detection ──▶ whisper.cpp (Metal) ──▶ claude -p --resume
                                                          │
speakers ◀── say / Kokoro ◀── lexicon ◀── speech lock ◀───┘
